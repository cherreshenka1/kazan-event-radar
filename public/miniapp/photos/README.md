# Mini App Photos

Сюда можно складывать локальные фотографии для карточек Mini App.

Правило именования файлов для каждой карточки:

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`

Пример:

- `photos/parks/black_lake/1.jpg`
- `photos/parks/black_lake/2.jpg`
- `photos/parks/black_lake/3.jpg`
- `photos/parks/black_lake/4.jpg`

Логика отображения:

- `1.jpg` становится обложкой карточки
- остальные файлы показываются в блоке `Подборка фото`
- если локальных фото пока нет, приложение использует резервное изображение
