# Автообновление раз в 3 дня

В проект добавлен отдельный workflow:

- `.github/workflows/refresh-content-every-3-days.yml`

Что он делает автоматически раз в 3 дня:

1. Обновляет афишу:
   - Yandex Afisha
   - MTS Live
   - Kassir, если есть рабочий auth state
   - официальный спорт
2. Пересобирает превью мероприятий.
3. Прогоняет аудит контента.
4. Перевыкладывает Mini App на GitHub Pages.

Каталог разделов обновляется отдельным workflow:

- `.github/workflows/refresh-catalog.yml`

Он запускается регулярно, но работает в режиме `--stale-only`: каждый раздел обновляется только тогда, когда истёк его собственный срок свежести.

Каждые 3 дня:

- еда
- экскурсии
- пешие маршруты
- активный отдых
- мастер-классы
- поездки на машине

Раз в 7 дней:

- парки
- достопримечательности
- отели

## Когда запускается

- афиша: раз в 3 дня
- каталог: workflow запускается регулярно, а частота конкретного раздела задаётся в `config/catalog-sources.json`
- по расписанию GitHub Actions
- вручную через вкладку `Actions`, если нужно запустить вне очереди

## Что нужно для полностью автоматической работы

В GitHub Secrets должны быть заполнены:

- `CLOUDFLARE_API_TOKEN`
- `CLOUDFLARE_ACCOUNT_ID`
- `YANDEX_AUTH_STATE_B64`
- `KASSIR_AUTH_STATE_B64` — желательно, но если его нет или он истёк, workflow не упадёт полностью, а просто пропустит Kassir

## Что обновляется без ручной помощи

Без ручной помощи у нас уже могут обновляться:

- события
- каталог разделов
- превью мероприятий
- live Mini App на GitHub Pages

## Что всё ещё может иногда требовать человека

Самое слабое место — `Kassir`.

Иногда он включает anti-bot проверку, и тогда нужно:

1. заново прогреть браузерный профиль локально
2. сохранить новый `kassir-state.json`
3. обновить `KASSIR_AUTH_STATE_B64` в GitHub Secrets

Остальная цепочка после этого снова работает автоматически.

## Как запускать вручную

Через GitHub:

1. Открой `Actions`
2. Выбери `Refresh Content And Deploy Mini App`
3. Нажми `Run workflow`

Можно отдельно указать:

- включать ли Kassir
- обновлять ли каталог в этом прогоне
