# Модерация карточек и фото

Этот процесс нужен, чтобы карточки Mini App заполнялись почти автоматически, но финальное решение оставалось за модератором.

## Что делает автоматика

1. Берёт текущие карточки из каталога.
2. Проверяет источники, которые уже привязаны к карточкам.
3. Собирает короткие черновики описаний.
4. Скачивает фото-кандидаты в закрытую папку `data/catalog-moderation/photo-candidates`.
5. Создаёт рабочую доску `data/catalog-moderation/review-board.md`.

Фото из `data` не попадают в Mini App сами. Это важно: сначала человек проверяет, нет ли чужого логотипа, водяного знака, рекламной плашки или нерелевантной картинки.

## Как запустить сбор кандидатов

```powershell
npm run catalog:moderation:candidates
```

После сборки можно открыть визуальную галерею одной командой:

```powershell
npm run catalog:moderation:open
```

Для короткой проверки можно ограничить количество карточек:

```powershell
npm run catalog:moderation:candidates -- --limit-per-section=1 --max-images-per-item=1
```

## Как одобрить фото

1. Откройте `data/catalog-moderation/review-gallery.html` или запустите `npm run catalog:moderation:open`.
2. Отметьте галочками только чистые и релевантные изображения.
3. Нажмите `Export approvals JSON`.
4. Сохраните результат как `config/catalog-moderation-approvals.json`.
5. Перед реальным применением можно проверить план:

```powershell
npm run catalog:moderation:dry-run
```

6. Если отчёт без ошибок, запустите:

```powershell
npm run catalog:moderation:apply
```

Скрипт перенесёт одобренные изображения в `public/miniapp/photos/<section>/<card>/1.jpg` или похожий файл с исходным расширением, затем обновит фото-манифест.

## Юридическая модель

Мы не копируем большие описания источников. В карточках должны быть свои короткие тексты, факты, логистика и ссылка на источник.

Фото нельзя одобрять, если на нём есть:

- водяной знак;
- логотип чужого сервиса;
- рекламная плашка другого агрегатора;
- неочевидные права;
- нерелевантная картинка.

Если фото сомнительное, лучше оставить карточку без него и подобрать другое вручную.
