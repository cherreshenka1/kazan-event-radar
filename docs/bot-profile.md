# Kazan Event Radar: профиль бота

## Name

KazanEventRadar

## About

Афиша, маршруты, места и напоминания по Казани в одном мини-приложении.

## Description / What can this bot do?

Kazan Event Radar помогает не пропустить интересную Казань: мероприятия, экскурсии, пешие маршруты, парки, рестораны, отели, секретные места и панорамы города.

В мини-приложении можно будет:
- смотреть актуальную афишу и подборки мест;
- добавлять события и локации в избранное;
- собирать личный план поездки;
- ставить напоминания за сутки и за час до мероприятия;
- открывать маршрут до места;
- переходить к покупке билетов на разных площадках;
- получать обновления и превью постов из канала.

## Short Welcome Message

Привет! Я Kazan Event Radar. Собираю для тебя лучшие события, маршруты, парки, рестораны, отели и секретные места Казани. Открой меню, добавь интересное в избранное и не забудь поставить напоминания.

## Compact Welcome Message

Открой Казань без лишнего поиска: афиша, маршруты, парки, еда, отели, билеты и напоминания в одном месте.

## Avatar Concept

Темный фон ночной Казани, зеленый радарный круг, буква K в центре, тонкая линия маршрута и силуэт купола/башни. Иконка должна хорошо читаться в маленьком круглом Telegram-аватаре.

## Welcome Picture Concept

Широкая карточка 640x360: вечерняя Казань, радарная сетка, силуэт Кремля и Кул Шариф, маршрутные точки, карточки “Афиша”, “Маршруты”, “Места”. Настроение: современный городской гид, аккуратно, премиально, без перегруза.
